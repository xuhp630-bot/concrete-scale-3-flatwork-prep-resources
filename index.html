A flatwork index for checking prep, quantity, finish, and project scope.

Flatwork Prep Sequence

  1. Start with the project type, dimensions, units, and site constraints.
  2. Separate quantity, delivery, base prep, access, labor, and risk assumptions.
  3. Use the linked references to compare scenarios before buying materials.
  4. Confirm final scope with local suppliers, contractors, or inspectors.
